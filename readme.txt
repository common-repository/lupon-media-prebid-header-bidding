== Description ==

Lupon Media Prebid - Header Bidding wrapper is for the publishers working with the Lupon Media in order to implement the ads with only a few clicks.
Ads.txt Content - include the ads.txt file from Lupon Media
Header Content - include the Lupon Media's Wrapper between the <head></head> part of your site
Footer Content - include the ad code for Premium ads from Lupon Media
Post Content - include standard display, video and native ads from Lupon Media inside the article after Paragraph #1,#2,#3,#4 and #5

== Installation ==

Go to plugins on your Wordpress. Click on Ad new and search for Lupon Media Prebid - Header Bidding. Once you see the Plugin then click on Install. 
After it is installed click on activate and you will find the plugin on the left side of your Wordpress as Lupon Media Prebid - Header Bidding

== Frequently Asked Questions ==

1. Where can I get the codes to include them in each box?
Lupon Media team will provide you all the necessary information with the login to the platform. Once logged in, you will be able to make the setup for the codes.
2. Where can I see the Revenue from those ads?
Lupon Media will provide you with the access to their platform so you can see your revenue after the tags are active.
